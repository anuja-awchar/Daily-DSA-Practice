## How to Contribute (Daily Workflow)

1. **Pick a problem** from the list above
2. **Solve it in Java** - write clean, commented code
3. **Create a file** in the appropriate topic folder (e.g., `arrays/TwoSum.java`)
4. **Commit** with a descriptive message: `feat: solved Two Sum - Arrays`
5. **Push** to GitHub
6. **Mark as done** by checking the box in this README

---

## Daily Commit Habit Tips

- Solve **at least 1 problem per day** - consistency > quantity
- Use **LeetCode / GeeksforGeeks / Codeforces** for problems
- Time yourself - aim for **20-30 minutes per problem**
- If stuck for >30 mins, check hints, then re-solve alone next day
- **Weekends:** Do 2-3 problems + revise the week's concepts
- Track your **streak** - don't break the chain!
